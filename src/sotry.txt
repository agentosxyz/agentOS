    addTerminalLog({ module: 'SMC', message: `Scanning liquidity pools at ${Math.random() * 64000}...` });

  const chartInterval = setInterval(() => {
    updateChartPrice(64500 + (Math.random() * 100 - 50));

    updatePnl(Math.random() * 500 - 100);

    clearInterval(pnlInterval);
  }
}, [isAgentActive])

// This schema guarantees fast writes for high-frequency AI logs and trades.

generator client {

3.2 The NestJS Module Pipeline (The Brain)

The backend must be structured into three distinct micro-services (Modules) that pass data linearly.

MarketIngestionModule: Connects to the Bitget WebSocket API. It pushes 1-second candle updates into a local Redis cache. Do not write raw price ticks to Postgres or the database will crash.

AIAnalyzerModule: Runs a cron job every 5 seconds. It reads the last 100 candles from Redis, passes them through your AI API prompt (e.g., "Identify Order Blocks in this OHLCV data array"), and calculates SMC.

RiskExecutionModule: When the AI finds an Order Block, this module intercepts it. It pulls the user's riskPerTrade from Postgres. If the trade requires $500 risk but the user only allowed $200, the module throws an AgentLog (WARNING) and rejects the trade. If it passes, it executes.

VOLUME 4: THE 3D ENGINE & MATH MATRICES (React Three Fiber)

To make the UI feel like a AAA video game, the 3D canvas cannot just be static. It must mathematically react to the market data flowing in from Volume 2 and Volume 3.

4.1 Volatility-Driven Lighting Math

When the market goes crazy, the lighting in the 3D room should subtly flash to create tension. We bind a <pointLight> intensity directly to the WebSocket price volatility.

// Inside your AgentScene.tsx using useFrame
import { useFrame } from '@react-three/fiber';
import { useRef } from 'react';
import * as THREE from 'three';
import { useDashboardState } from '../store/useDashboardState';

export function DynamicMonitorGlow() {
  const lightRef = useRef();
  const currentPrice = useDashboardState(state => state.currentPrice);
  const previousPrice = useRef(currentPrice);

  useFrame(() => {
    if (!lightRef.current) return;
    
    // Calculate the delta (volatility) between frames
    const delta = Math.abs(currentPrice - previousPrice.current);
    
    // Base intensity + volatility multiplier
    // A normal tick changes intensity slightly. A massive spike causes a flash.
    const targetIntensity = 0.5 + (delta * 0.1); 
    
    // Smooth interpolation (Lerp) so the light doesn't strobe aggressively
    lightRef.current.intensity = THREE.MathUtils.lerp(
      lightRef.current.intensity, 
      Math.min(targetIntensity, 3.0), // Cap max intensity at 3.0
      0.1
    );

    previousPrice.current = currentPrice;
  });

  return <pointLight ref={lightRef} position={[0, 1, 1]} color="#A3E635" />;
}


4.2 Camera Quaternion Slerping (Cinematic Transitions)

When the user clicks "Deploy Agent," the camera should smoothly sweep from a wide "Idle" shot to a close-up over-the-shoulder "Typing" shot.

Idle Position: [0, 1.5, 4] (Looking at the desk)

Active Position: [1, 2, 2] (Looking over the agent's shoulder at the screen)

Using useFrame and THREE.Vector3().lerp(), you transition the camera over 1.5 seconds triggered by the isAgentActive boolean. This prevents jarring jump-cuts.

4.3 Animation Blending (Mixamo Crossfades)

Never use .play() directly without crossfading, or the character's skeleton will snap violently, breaking the illusion.
Using @react-three/drei's useAnimations:

useEffect(() => {
  if (isAgentActive) {
    actions['Sitting Idle'].fadeOut(0.5);
    actions['Heavy Typing'].reset().fadeIn(0.5).play();
  } else {
    actions['Heavy Typing'].fadeOut(0.5);
    actions['Sitting Idle'].reset().fadeIn(0.5).play();
  }
}, [isAgentActive, actions]);


VOLUME 5: WEB3 INTEGRATION & DEPLOYMENT STRATEGY

Hackathon judges deduct massive points if your project only runs on localhost:3000. You must deploy this as a live, secure Web3 dApp.

5.1 The Web3 Wallet Layer (Top Right Header)

You must implement RainbowKit combined with Wagmi for the connection state.

Wrap your Next.js _app.tsx in <WagmiConfig>.

Map the "Not Connected" button in Zone 1 to the RainbowKit ConnectButton.

The Auth Flow: When a user connects their wallet, trigger a cryptographic signature (signMessage). Send that signature to the NestJS backend to authenticate the WebSocket connection. Do not let unauthenticated sockets listen to the AI stream.

5.2 Hackathon Mock Vault (Smart Contract Concept)

While this is a "Paper" trading demo, you need a Web3 angle for Bitget.

The Pitch: Explain to the judges that in production, the "Allocated Capital" isn't just a number in a database. It is a deposit into a User-Specific Smart Contract Vault.

The Logic: The user deposits USDT into their Vault. The Agent OS backend has a restricted API key that is only authorized to execute trades using the Vault's liquidity. The backend cannot withdraw the funds, only the user can. This creates a trustless autonomous system.

5.3 High-Performance Deployment Architecture

Do not deploy the frontend and backend on the same server. WebSockets will choke the Next.js render cycle.

Frontend (Next.js): Deploy to Vercel. It provides edge caching for the massive 3D .glb files.

Backend (NestJS): Deploy to Railway.app or Render.com. These handle persistent WebSocket connections (Socket.io) infinitely better than Serverless functions.

Database (PostgreSQL): Deploy via Supabase. It is blisteringly fast and provides a generous free tier for the hackathon.

Storage: Ensure agent_os.glb is compressed using Draco compression (npx gltf-pipeline -i agent_os.glb -o agent_draco.glb -d) before uploading to Vercel, reducing the file size from ~15MB to ~2MB.

6.5 The Internal Event Bus

@nestjs/event-emitter is the spine. Every cross-module signal goes through it
so no service ever imports another service's internal class. Canonical events:

  bitget.tick            { symbol, price, ts, source }
  bitget.candle          { symbol, o, h, l, c, v, ts }
  bitget.degraded        { reason }
  bitget.restored        {}
  perception.tick        { symbol, price, ts, source }
  perception.candle      { symbol, candle }
  smc.pattern            { sessionId, patternType, direction, priceLevel, asset }
  risk.signal            { sessionId, side, entry, sl, tp, sizeUsdt, leverage }
  risk.killswitch        { sessionId, reason }
  execution.opened       { sessionId, trade }
  execution.closed       { sessionId, trade }
  portfolio.snapshot     { sessionId, unrealized, realized, winRate, openPositions }

Rule: payloads are typed, immutable, and never carry class instances — only
JSON-safe values. This keeps the bus serializable for future Redis pub/sub
sharding without rewrites.

7.3 BitgetWsService — full skeleton

@Injectable()
export class BitgetWsService implements OnModuleInit, OnModuleDestroy {
  private ws?: WebSocket;
  private retryAttempt = 0;
  private lastTickAt = 0;
  private lastPrice = 0;
  private degraded = false;
  private syntheticTimer?: NodeJS.Timeout;
  private heartbeatTimer?: NodeJS.Timeout;
  private watchdogTimer?: NodeJS.Timeout;
  private readonly symbol: string;

  constructor(
    private readonly cfg: ConfigService,
    private readonly bus: EventEmitter2,
    private readonly logger: PinoLogger,
  ) {
    this.logger.setContext('BitgetWsService');
    this.symbol = cfg.get('BITGET_DEFAULT_SYMBOL');
  }

  onModuleInit()    { this.connect(); this.startWatchdog(); }
  onModuleDestroy() { this.teardown(); }

  private connect() {
    const url = this.cfg.get<string>('BITGET_WS_URL');
    this.ws = new WebSocket(url);
    this.ws.on('open',    () => this.onOpen());
    this.ws.on('message', (d) => this.onMessage(d.toString()));
    this.ws.on('close',   () => this.scheduleReconnect());
    this.ws.on('error',   (e) => this.logger.warn({ err: e }, 'ws error'));
  }

  private onOpen() {
    this.retryAttempt = 0;
    this.ws!.send(JSON.stringify({
      op: 'subscribe',
      args: [
        { instType: 'USDT-FUTURES', channel: 'ticker',   instId: this.symbol },
        { instType: 'USDT-FUTURES', channel: 'candle1m', instId: this.symbol },
      ],
    }));
    this.heartbeatTimer = setInterval(() => this.ws?.send('ping'), 25_000);
    if (this.degraded) {
      this.degraded = false;
      this.stopSynthetic();
      this.bus.emit('bitget.restored', {});
      this.logger.info('upstream restored');
    }
  }

  private onMessage(raw: string) {
    if (raw === 'pong') return;
    let msg: any; try { msg = JSON.parse(raw); } catch { return; }
    if (msg.event === 'subscribe') return;
    const arg = msg.arg;
    const data = msg.data?.[0];
    if (!arg || !data) return;

    if (arg.channel === 'ticker') {
      const price = Number(data.lastPr);
      if (!Number.isFinite(price)) return;
      this.lastTickAt = Date.now();
      this.lastPrice = price;
      this.bus.emit('bitget.tick', {
        symbol: arg.instId,
        price,
        ts: Date.now(),
        source: 'bitget',
      });
    } else if (arg.channel === 'candle1m') {
      // candle1m payload: [ts, o, h, l, c, vol, usdtVol]
      const [ts, o, h, l, c, v] = data.map(Number);
      this.bus.emit('bitget.candle', {
        symbol: arg.instId,
        o, h, l, c, v, ts,
      });
    }
  }

  private scheduleReconnect() {
    clearInterval(this.heartbeatTimer);
    const delay = Math.min(30_000, 1_000 * 2 ** this.retryAttempt++);
    this.logger.warn(`reconnecting in ${delay}ms (attempt ${this.retryAttempt})`);
    setTimeout(() => this.connect(), delay);
  }

  private startWatchdog() {
    const failoverMs = this.cfg.get<number>('BITGET_FAILOVER_MS');
    this.watchdogTimer = setInterval(() => {
      if (this.lastTickAt === 0) return;
      const gap = Date.now() - this.lastTickAt;
      if (gap > failoverMs && !this.degraded) {
        this.degraded = true;
        this.bus.emit('bitget.degraded', { reason: `no tick for ${gap}ms` });
        this.startSynthetic(this.lastPrice || 64_000);
        this.logger.warn('upstream degraded, synthetic feed engaged');
      }
    }, 1_000);
  }

  private startSynthetic(seed: number) {
    let price = seed;
    this.syntheticTimer = setInterval(() => {
      const drift = (Math.random() - 0.5) * (price * 0.0008);
      price = Math.max(1, price + drift);
      this.bus.emit('bitget.tick', {
        symbol: this.symbol, price, ts: Date.now(), source: 'synthetic',
      });
      // synthetic 1m candle assembled every 60s using observed highs/lows
      // (simplified: emit a candle every 5s for demo cadence)
    }, 1_000);
  }

  private stopSynthetic() { clearInterval(this.syntheticTimer!); }
  private teardown() {
    clearInterval(this.heartbeatTimer);
    clearInterval(this.watchdogTimer);
    this.stopSynthetic();
    this.ws?.close();
  }
}

7.4 BitgetRestService — seed candles

On boot, before any browser is allowed to start an engine, the SMC buffer is
seeded with 200 real 1-minute candles so detection runs immediately.

  GET {BITGET_REST_URL}/api/v2/spot/market/candles
       ?symbol=BTCUSDT&granularity=1min&limit=200
  Response: array of [ts, o, h, l, c, baseVol, quoteVol].

7.5 PerceptionService

@Injectable()
export class PerceptionService implements OnModuleInit {
  private lastLoggedAt = new Map<string, number>(); // symbol -> ts

  constructor(
    private readonly bus: EventEmitter2,
    private readonly redis: RedisService,
    private readonly gateway: AgentGateway,
  ) {}

  onModuleInit() {
    this.bus.on('bitget.tick',     (p) => this.onTick(p));
    this.bus.on('bitget.candle',   (p) => this.onCandle(p));
    this.bus.on('bitget.degraded', () => this.broadcastDegraded(true));
    this.bus.on('bitget.restored', () => this.broadcastDegraded(false));
  }

  private async onTick(p: { symbol; price; ts; source }) {
    await this.redis.setPrice(p.symbol, p.price);
    this.bus.emit('perception.tick', p);
    this.gateway.broadcast('market_tick', {
      symbol: p.symbol, price: p.price, timestamp: Math.floor(p.ts / 1000),
    });

    // Rate-limited terminal log: max 1 per symbol per 2s
    const last = this.lastLoggedAt.get(p.symbol) ?? 0;
    if (p.ts - last > 2_000) {
      this.gateway.broadcastLog('PERC', `Tick ${p.symbol} @ ${p.price.toFixed(2)}`, 'INFO');
      this.lastLoggedAt.set(p.symbol, p.ts);
    }
  }

  private onCandle(p: any) {
    this.bus.emit('perception.candle', p);
  }

  private broadcastDegraded(degraded: boolean) {
    this.gateway.broadcastLog(
      'PERC',
      degraded ? 'Upstream degraded — synthetic feed engaged.' : 'Upstream restored.',
      degraded ? 'WARNING' : 'SUCCESS',
    );
  }
}

VOLUME 8: SMC ENGINE — THE BRAIN

8.1 CandleBuffer (ring buffer, typed)

class CandleBuffer {
  private buf: Candle[];
  private writeIdx = 0;
  private filled = false;
  constructor(private readonly capacity = 200) { this.buf = new Array(capacity); }

  push(c: Candle) {
    this.buf[this.writeIdx] = c;
    this.writeIdx = (this.writeIdx + 1) % this.capacity;
    if (this.writeIdx === 0) this.filled = true;
  }

  all(): Candle[] {
    if (!this.filled) return this.buf.slice(0, this.writeIdx);
    return [...this.buf.slice(this.writeIdx), ...this.buf.slice(0, this.writeIdx)];
  }

  last(): Candle | undefined { return this.all().at(-1); }
}

One CandleBuffer per symbol, kept in a Map<symbol, CandleBuffer> inside
SmcService. For v1, only BTCUSDT is tracked.

8.2 Detector: Order Block

export function detectOrderBlock(buf: Candle[]): OrderBlockSignal | null {
  if (buf.length < 20) return null;
  const atr = avgTrueRange(buf, 14);                  // standard ATR
  const last = buf.at(-1)!;
  const candidate = buf.at(-2)!;
  const impulseRange = Math.abs(last.c - last.o);
  if (impulseRange < 1.5 * atr) return null;          // not impulsive enough

  const lastBull = last.c > last.o;
  const candBull = candidate.c > candidate.o;
  // Bullish OB: impulsive bull candle preceded by a bear candle
  if (lastBull && !candBull) {
    return {
      patternType: 'ORDER_BLOCK',
      direction: 'BULLISH',
      priceLevel: round2((candidate.h + candidate.l) / 2),
      lowBound: candidate.l, highBound: candidate.h, ts: candidate.ts,
    };
  }
  if (!lastBull && candBull) {
    return {
      patternType: 'ORDER_BLOCK',
      direction: 'BEARISH',
      priceLevel: round2((candidate.h + candidate.l) / 2),
      lowBound: candidate.l, highBound: candidate.h, ts: candidate.ts,
    };
  }
  return null;
}

8.3 Detector: Break of Structure

export function detectBoS(buf: Candle[]): BoSSignal | null {
  if (buf.length < 12) return null;
  const window = buf.slice(-11, -1);                  // last 10 closed candles
  const swingHigh = Math.max(...window.map(c => c.h));
  const swingLow  = Math.min(...window.map(c => c.l));
  const last = buf.at(-1)!;
  if (last.c > swingHigh) return { patternType:'BOS', direction:'BULLISH', priceLevel: round2(swingHigh), ts: last.ts };
  if (last.c < swingLow)  return { patternType:'BOS', direction:'BEARISH', priceLevel: round2(swingLow),  ts: last.ts };
  return null;
}

8.4 Detector: Fair Value Gap

export function detectFVG(buf: Candle[]): FVGSignal | null {
  if (buf.length < 3) return null;
  const a = buf.at(-3)!, b = buf.at(-2)!, c = buf.at(-1)!;
  // bullish FVG: a.high < c.low
  if (a.h < c.l) return { patternType:'FVG', direction:'BULLISH', priceLevel: round2((a.h + c.l)/2), ts: b.ts };
  if (a.l > c.h) return { patternType:'FVG', direction:'BEARISH', priceLevel: round2((a.l + c.h)/2), ts: b.ts };
  return null;
}

8.5 SmcService — orchestration

@Injectable()
export class SmcService implements OnModuleInit {
  private buffers = new Map<string, CandleBuffer>();
  // de-dup: roundedPrice|direction|patternType -> lastEmittedAt
  private recentEmits = new Map<string, number>();
  private readonly dedupTtl = 5 * 60_000;             // 5 minutes

  constructor(
    private readonly bus: EventEmitter2,
    private readonly sessions: SessionService,
    private readonly gateway: AgentGateway,
  ) {}

  onModuleInit() {
    this.bus.on('perception.candle', (p) => this.onCandle(p));
  }

  seedCandles(symbol: string, candles: Candle[]) {
    const buf = new CandleBuffer(200);
    candles.forEach(c => buf.push(c));
    this.buffers.set(symbol, buf);
  }

  private onCandle({ symbol, ...c }: any) {
    let buf = this.buffers.get(symbol);
    if (!buf) { buf = new CandleBuffer(200); this.buffers.set(symbol, buf); }
    buf.push(c);
    const all = buf.all();

    for (const detector of [detectOrderBlock, detectBoS, detectFVG]) {
      const sig = detector(all);
      if (!sig) continue;
      if (!this.isFresh(symbol, sig)) continue;
      this.emitForActiveSessions(symbol, sig);
    }
  }

  private isFresh(symbol: string, sig: { patternType; direction; priceLevel }): boolean {
    const key = `${symbol}|${sig.patternType}|${sig.direction}|${Math.round(sig.priceLevel)}`;
    const now = Date.now();
    const last = this.recentEmits.get(key) ?? 0;
    if (now - last < this.dedupTtl) return false;
    this.recentEmits.set(key, now);
    return true;
  }

  private emitForActiveSessions(symbol: string, sig: any) {
    const active = this.sessions.allActive();
    for (const session of active) {
      const payload = {
        sessionId: session.id, asset: symbol,
        patternType: sig.patternType, direction: sig.direction,
        priceLevel: sig.priceLevel,
      };
      this.bus.emit('smc.pattern', payload);
      this.gateway.emitToSession(session.id, 'smc_pattern_detected', {
        patternType: sig.patternType, direction: sig.direction,
        priceLevel: sig.priceLevel, asset: symbol,
      });
      this.gateway.emitLogToSession(session.id, 'SMC',
        `Identified ${sig.direction} ${sig.patternType} on ${symbol} at $${sig.priceLevel}`,
        'INFO');
    }
  }
}

8.6 Why three detectors are enough

OB gives the entry zone, BoS confirms market structure shifted in our favor,
FVG gives an extra confluence the agent can mention in its reasoning. Risk
only acts on Order Blocks; BoS and FVG are emitted purely for terminal flavor.
This keeps the trade rate honest (~1 trade every 1–3 minutes on BTCUSDT 1m).

9.1 RiskService — sizing

@Injectable()
export class RiskService implements OnModuleInit {
  constructor(
    private readonly bus: EventEmitter2,
    private readonly sessions: SessionService,
    private readonly redis: RedisService,
    private readonly gateway: AgentGateway,
  ) {}

  onModuleInit() {
    this.bus.on('smc.pattern', (p) => this.onPattern(p));
  }

  private async onPattern(p: any) {
    if (p.patternType !== 'ORDER_BLOCK') return;
    const session = this.sessions.getById(p.sessionId);
    if (!session || !session.isActive) return;

    const price = await this.redis.getPrice(p.asset);
    if (!price) return;

    const side = p.direction === 'BULLISH' ? 'LONG' : 'SHORT';
    const entry = price;
    const sl = side === 'LONG' ? entry * 0.99 : entry * 1.01;
    const tp = side === 'LONG' ? entry * 1.02 : entry * 0.98;

    const { sizeUsdt, leverage } = this.calculatePositionSize({
      allocatedCapital: session.allocatedCapital,
      riskPerTrade: session.riskPerTrade,
      entryPrice: entry, stopLoss: sl,
    });

    if (sizeUsdt < 10) {
      this.gateway.emitLogToSession(session.id, 'RISK',
        `Rejected ${side}: size $${sizeUsdt.toFixed(2)} below minimum.`,
        'WARNING');
      return;
    }

    this.bus.emit('risk.signal', {
      sessionId: session.id, symbol: p.asset,
      side, entry, sl, tp, sizeUsdt, leverage,
    });
  }

  calculatePositionSize(i: {
    allocatedCapital: number; riskPerTrade: number;
    entryPrice: number; stopLoss: number;
  }) {
    const riskUsdt  = i.allocatedCapital * (i.riskPerTrade / 100);
    const stopDist  = Math.abs(i.entryPrice - i.stopLoss) / i.entryPrice;
    const sizeUsdt  = stopDist === 0 ? 0 : riskUsdt / stopDist;
    const leverage  = Math.max(1, Math.min(20, Math.round(sizeUsdt / i.allocatedCapital)));
    return { sizeUsdt: round2(sizeUsdt), leverage };
  }

  checkDrawdown(sessionId: string, cumulativePnl: number): boolean {
    const s = this.sessions.getById(sessionId);
    if (!s) return false;
    const threshold = -(s.allocatedCapital * s.maxDrawdown / 100);
    if (cumulativePnl <= threshold) {
      this.bus.emit('risk.killswitch', { sessionId, reason: 'MAX_DRAWDOWN' });
      this.gateway.emitLogToSession(sessionId, 'RISK',
        `Hard stop hit at ${cumulativePnl.toFixed(2)}. Halting agent.`,
        'CRITICAL');
      return true;
    }
    return false;
  }
}

9.2 ExecutionService — paper trades

@Injectable()
export class ExecutionService implements OnModuleInit {
  private slTpTimer?: NodeJS.Timeout;

  constructor(
    private readonly bus: EventEmitter2,
    private readonly prisma: PrismaService,
    private readonly redis: RedisService,
    private readonly sessions: SessionService,
    private readonly ledger: LedgerService,
    private readonly gateway: AgentGateway,
  ) {}

  onModuleInit() {
    this.bus.on('risk.signal',     (p) => this.openPaperTrade(p));
    this.bus.on('risk.killswitch', (p) => this.closeAllForSession(p.sessionId, 'KILLSWITCH'));
    this.slTpTimer = setInterval(() => this.checkSlTpForAllSessions(), 500);
  }

  private async openPaperTrade(sig: any) {
    const open = this.ledger.getOpen(sig.sessionId);
    if (open.length >= 5) {
      this.gateway.emitLogToSession(sig.sessionId, 'EXEC',
        'Rejected: max 5 open positions reached.', 'WARNING');
      return;
    }

    const trade = await this.prisma.tradeExecution.create({
      data: {
        sessionId: sig.sessionId,
        symbol: sig.symbol, side: sig.side,
        sizeUsdt: sig.sizeUsdt, leverage: sig.leverage,
        entryPrice: sig.entry, stopLoss: sig.sl, takeProfit: sig.tp,
        status: 'OPEN',
      },
    });

    this.ledger.add(sig.sessionId, trade);
    this.bus.emit('execution.opened', { sessionId: sig.sessionId, trade });

    this.gateway.emitToSession(sig.sessionId, 'trade_execution', {
      tradeId: trade.id, symbol: trade.symbol, side: trade.side,
      sizeUsdt: trade.sizeUsdt, leverage: trade.leverage,
      entryPrice: trade.entryPrice, stopLoss: trade.stopLoss, takeProfit: trade.takeProfit,
    });
    this.gateway.emitLogToSession(sig.sessionId, 'EXEC',
      `Opened ${trade.side} ${trade.symbol} @ ${trade.entryPrice.toFixed(2)}`,
      'SUCCESS');
  }

  async closePaperTrade(tradeId: string, exitPrice: number, reason: string) {
    const trade = await this.prisma.tradeExecution.findUnique({ where: { id: tradeId } });
    if (!trade || trade.status !== 'OPEN') return;

    const direction = trade.side === 'LONG' ? 1 : -1;
    const pnl = direction * (exitPrice - trade.entryPrice) * (trade.sizeUsdt / trade.entryPrice);

    const closed = await this.prisma.tradeExecution.update({
      where: { id: tradeId },
      data: { status: 'CLOSED', realizedPnl: round2(pnl), closedAt: new Date() },
    });

    this.ledger.remove(trade.sessionId, tradeId);
    this.bus.emit('execution.closed', { sessionId: trade.sessionId, trade: closed });

    const sev = pnl >= 0 ? 'SUCCESS' : 'WARNING';
    this.gateway.emitLogToSession(trade.sessionId, 'EXEC',
      `Closed ${trade.side} ${trade.symbol} (${reason}) PnL ${pnl >= 0 ? '+' : ''}${pnl.toFixed(2)}`,
      sev);
  }

  async checkSlTpForAllSessions() {
    const activeSessions = this.sessions.allActive();
    for (const s of activeSessions) {
      const open = this.ledger.getOpen(s.id);
      if (open.length === 0) continue;
      const price = await this.redis.getPrice(open[0].symbol);
      if (!price) continue;
      for (const t of open) {
        const hitSl = t.side === 'LONG' ? price <= t.stopLoss   : price >= t.stopLoss;
        const hitTp = t.side === 'LONG' ? price >= t.takeProfit : price <= t.takeProfit;
        if (hitSl) await this.closePaperTrade(t.id, t.stopLoss,   'SL');
        else if (hitTp) await this.closePaperTrade(t.id, t.takeProfit, 'TP');
      }
    }
  }

  async closeAllForSession(sessionId: string, reason: string) {
    const open = this.ledger.getOpen(sessionId);
    if (open.length === 0) return;
    const price = await this.redis.getPrice(open[0].symbol);
    if (!price) return;
    for (const t of open) await this.closePaperTrade(t.id, price, reason);
  }
}

9.4 PortfolioService — 2-second aggregator

@Injectable()
export class PortfolioService {
  constructor(
    private readonly prisma: PrismaService,
    private readonly redis: RedisService,
    private readonly sessions: SessionService,
    private readonly ledger: LedgerService,
    private readonly risk: RiskService,
    private readonly gateway: AgentGateway,
  ) {}

  @Interval(2_000)
  async tickPortfolios() {
    for (const s of this.sessions.allActive()) {
      const open = this.ledger.getOpen(s.id);
      const price = open.length > 0 ? await this.redis.getPrice(open[0].symbol) : 0;

      let unrealized = 0;
      for (const t of open) {
        if (!price) continue;
        const dir = t.side === 'LONG' ? 1 : -1;
        unrealized += dir * (price - t.entryPrice) * (t.sizeUsdt / t.entryPrice);
      }

      const closedAgg = await this.prisma.tradeExecution.groupBy({
        by: ['status'],
        where: { sessionId: s.id, status: 'CLOSED' },
        _sum: { realizedPnl: true },
        _count: true,
      });
      const wins = await this.prisma.tradeExecution.count({
        where: { sessionId: s.id, status: 'CLOSED', realizedPnl: { gt: 0 } },
      });
      const closedTotal = closedAgg[0]?._count ?? 0;
      const realized = Number(closedAgg[0]?._sum.realizedPnl ?? 0);
      const winRate = closedTotal === 0 ? 0 : round2((wins / closedTotal) * 100);
      const cumulative = realized + unrealized;

      this.gateway.emitToSession(s.id, 'portfolio_update', {
        unrealizedPnl: round2(unrealized),
        realizedPnl: round2(realized),
        winRate,
        openPositions: open.length,
      });

      this.risk.checkDrawdown(s.id, cumulative);
    }
  }

  @Cron('*/30 * * * * *')
  async persistSnapshot() {
    // optional time-series row for post-demo chart; skip if no active session
  }
}

VOLUME 10: AUTH, GATEWAY, OBSERVABILITY & RUNBOOK

10.1 SIWE-style Wallet Auth

The frontend's wallet signature is the only credential. Flow:

  POST /auth/nonce { walletAddress }
    -> { nonce: <random 32-byte hex>, message: 'Sign to enter Agent.OS: <nonce>' }
    -> server stores nonce in Redis with EX = NONCE_TTL_SECONDS (default 300s)
  POST /auth/verify { walletAddress, signature }
    -> recover signer with viem.verifyMessage; if !== walletAddress -> 401
    -> upsert User row, issue JWT (HS256, JWT_SECRET, exp = JWT_TTL_SECONDS)
    -> { token, userId }

The browser then opens its Socket.IO connection with:
  io(BACKEND_URL, { auth: { token }, transports: ['websocket'] });

10.2 Gateway Auth Hook

@WebSocketGateway({ cors: { ... }, transports: ['websocket'] })
export class AgentGateway implements OnGatewayConnection, OnGatewayDisconnect {
  @WebSocketServer() server: Server;

  constructor(
    private readonly auth: AuthService,
    private readonly sessions: SessionService,
    private readonly logger: PinoLogger,
  ) {}

  async handleConnection(client: Socket) {
    try {
      const token = client.handshake.auth?.token;
      const userId = await this.auth.verifyJwt(token);
      const session = await this.sessions.createForSocket(client.id, userId);
      client.data.sessionId = session.id;
      client.join(`session:${session.id}`);
      client.emit('session_ready', { sessionId: session.id });
    } catch (e) {
      client.emit('agent_log', {
        timestamp: hhmmss(), module: 'SYSTEM',
        message: 'Auth failed — disconnecting.', type: 'CRITICAL',
      });
      client.disconnect(true);
    }
  }

  async handleDisconnect(client: Socket) {
    if (client.data.sessionId) await this.sessions.endById(client.data.sessionId);
  }

  @SubscribeMessage('START_ENGINE')
  async onStart(@ConnectedSocket() c: Socket, @MessageBody() dto: StartEngineDto) {
    await this.sessions.start(c.data.sessionId, dto);
    this.emitLogToSession(c.data.sessionId, 'SYSTEM', 'Engine started.', 'SUCCESS');
  }

  @SubscribeMessage('STOP_ENGINE')
  async onStop(@ConnectedSocket() c: Socket) {
    await this.sessions.pause(c.data.sessionId);
  }

  @SubscribeMessage('RESET')
  async onReset(@ConnectedSocket() c: Socket) {
    await this.sessions.reset(c.data.sessionId);
  }

  @SubscribeMessage('TERMINATE')
  async onTerminate(@ConnectedSocket() c: Socket) {
    await this.sessions.terminate(c.data.sessionId, 'TERMINATED_BY_USER');
  }

  // ---- outbound helpers used by every other service ----

  broadcast(event: string, payload: any) { this.server.emit(event, payload); }

  emitToSession(sessionId: string, event: string, payload: any) {
    this.server.to(`session:${sessionId}`).emit(event, payload);
  }

  broadcastLog(module: ModuleType, message: string, type: LogSeverity) {
    this.server.emit('agent_log', { timestamp: hhmmss(), module, message, type });
  }

  emitLogToSession(sessionId: string, module: ModuleType, message: string, type: LogSeverity) {
    this.emitToSession(sessionId, 'agent_log', { timestamp: hhmmss(), module, message, type });
  }
}

10.3 SessionService — full lifecycle

@Injectable()
export class SessionService {
  // in-memory hot cache for routing; Prisma is source of truth
  private bySocket = new Map<string, AgentSession>();
  private byId     = new Map<string, AgentSession>();

  constructor(private readonly prisma: PrismaService) {}

  async createForSocket(socketId: string, userId: string): Promise<AgentSession> {
    const session = await this.prisma.agentSession.create({
      data: {
        userId, allocatedCapital: 0, riskPerTrade: 1, maxDrawdown: 10,
        status: 'PAUSED',
      },
    });
    this.bySocket.set(socketId, session);
    this.byId.set(session.id, session);
    return session;
  }

  async start(sessionId: string, dto: StartEngineDto) {
    const s = await this.prisma.agentSession.update({
      where: { id: sessionId },
      data: {
        allocatedCapital: dto.allocatedCapital,
        riskPerTrade: dto.riskPerTrade,
        maxDrawdown: dto.maxDrawdown,
        status: 'ACTIVE',
      },
    });
    this.byId.set(sessionId, s);
  }

  async pause(sessionId: string)     { /* status: PAUSED */ }
  async reset(sessionId: string)     { /* close-all then status: PAUSED, zero capital */ }
  async terminate(id: string, why: SessionStatus) {
    await this.prisma.agentSession.update({
      where: { id }, data: { status: why, terminatedAt: new Date() },
    });
    this.byId.delete(id);
  }
  async endById(sessionId: string)   { await this.terminate(sessionId, 'TERMINATED_BY_USER'); }

  getById(id: string)   { return this.byId.get(id); }
  allActive(): AgentSession[] { return [...this.byId.values()].filter(s => s.status === 'ACTIVE'); }
}

10.4 Throttling

A custom ThrottleInterceptor on the gateway: max 30 inbound messages per socket
per 60 seconds (stored in Redis when available, in-memory Map otherwise).
Excess events ack with { ok: false, error: 'THROTTLED' } and emit a SYSTEM
agent_log of type WARNING. Prevents a misbehaving client from DoSing the bus.

10.7 Performance Budgets

  - Tick fan-out latency p95: < 250ms (Bitget -> browser).
  - SMC pass per new candle: < 5ms (200 candles, 3 detectors, all O(N)).
  - portfolio_update emit drift: 2s ± 100ms.
  - Memory RSS with 20 sockets, 1 active session: < 200MB.
  - CPU on a 1 vCPU Fly machine: < 15%.

10.8 Failure Modes & Behavior

  - Bitget WS disconnect          -> exponential reconnect, watchdog flips
                                     synthetic feed after 8s, log WARNING.
  - Bitget REST seed fails        -> log error, boot continues, SMC waits
                                     until first real candle accumulates.
  - Redis down                    -> in-mem LRU fallback; no user impact.
  - Postgres down                 -> /health returns ok:false; gateway still
                                     accepts connections but rejects START_ENGINE
                                     with WS ack { ok:false, error:'DB_DOWN' }.
  - Single trade write fails      -> emit agent_log WARNING, signal dropped,
                                     bus continues.
  - unhandledRejection            -> logged, NOT process.exit.
